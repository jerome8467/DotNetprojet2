# P2FixAnAppDotNetCode

Application e-commerce ASP.NET développée dans le cadre d'un projet de formation.

## Prérequis

- Visual Studio
- .NET 9.0 SDK

## Installation

1. Cloner le dépôt :
   git clone https://github.com/jerome8467/DotNetprojet2.git

2. Ouvrir le fichier `P2FixAnAppDotNetCode.sln` dans Visual Studio

3. Lancer l'application avec `F5`

## Tests

Le projet contient un projet de tests `P2FixAnAppDotNetCode.Tests`.
Pour lancer les tests, ouvrir l'explorateur de tests via le menu `Test` dans Visual Studio.
